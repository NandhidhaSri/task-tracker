
TASK TRACKER -FULL STACK PROJECT

OVERVIEW

    TASK TRACKER IS A "FULL STACK PROJECT" BUILT USING SPRING BOOT , MYSQL,AND HTML/CSS/JAVASCRIPT.
    IT ALLOWS USERS TO ADD,VIEW AND DELETE TASKS THROUGH A CLEAN,RESPONSIVE INTERFACE.
    THIS PROJECT DEMONSTRATES "BACKEND REST APIs INTEGRATION WITH FRONTEND ,MAKING IT A PERFECT FOR JAVA FULL STACK ROLE.

    FEATURES:
    ADD TASKS WITH A TITLE.
    VIEW ALL TASKS IN A LIST.
    DELETE TASKS INDIVIDUALLY.
    RESPONSIVE AND MODERN UI WITH STYLED BUTTONS AND LAYOUT .
    FULLY INTEGRATED " DPRING BOOT BACKEND " WITH "MYSQL DATABASE".
    SIMPLE ,EASY TO UNDERSTAND CODE STRUCTURE.
